## `Getting Started`

To get a local copy of this project:

Clone this repository or download the Zip folder:
```
git clone https://github.com/Amtrest/The-minimalist-place.git
```

Enter in root directory:
```
cd TheMinimalistPlace
```
Then run:
```
- bundle install
- npm install
```
Create Database (Mandatory)

```
rails db:create
rails db:schema:load
```

If you have made a migration the run this command

```
rails db:migrate
```

run TheMinimalistPlace app on server:
```
rails s
```

Open Source code of TheMinimalistPlace app on VS-Code

```
code .
```
Run tests

```
rspec
```
